PBR Shader Guide

Полное руководство по установке и настройке реалистичного PBR шейдера для Garry's Mod.

АХТУНГ! Важный момент

x64 версия GMod'а не поддерживается. Для работы шейдера используйте обычную версию игры, ветку pre-release или dev.

Как поставить эту красоту

Установка простая: скачайте и запустите .bat-файл. Он автоматически установит все необходимые модули для работы шейдера.

Что такое PBR?

PBR (Physically Based Rendering) — это современный шейдер, портированный из Strata Source, который делает освещение и материалы в игре значительно реалистичнее. Он работает как на моделях, так и на брашах (стенах, полах), используя набор специальных PBR-текстур.

Настройки в VMT-файле

Ниже перечислены основные параметры для настройки материалов в .vmt файлах.

Основа

  • $basetexture — Основная цветная текстура (Albedo).
  • $bumpmap — Карта нормалей для имитации рельефа.

PBR Карты

  • $mraotexture — Карта, содержащая (R) Metallic, (G) Roughness, (B) Ambient Occlusion.
  • $emissiontexture — Карта свечения. Должна быть цветной.

Параллакс

  • $parallax — Включает эффект глубины. Требует карту высот в альфа-канале нормалей.
  • $parallaxdepth — Глубина эффекта. Рекомендуется 0.1.

Прочее

  • $model 1 — Обязательный параметр для материалов на моделях.
  • $alphatestreference — Порог прозрачности для материалов вроде решеток или листвы.

ЭТО ВАЖНО

  1. Карты с PBR-шейдером запускайте только через консоль командой map имя_карты. Запуск через главное меню может привести к вылету.
  2. Для корректного освещения моделей введите в консоль r_PhysPropStaticLighting 0. Рекомендуется добавить эту команду в ваш autoexec.cfg.

Пример VMT-файла

"PBR"
{
    "$basetexture"      "models/example/gun_albedo"
    "$bumpmap"          "models/example/gun_normal"
    "$mraotexture"      "models/example/gun_mrao"
    "$emissiontexture"  "models/example/gun_emission"

    "$model"            "1"
    "$surfaceprop"      "metal"
}

Форматы Текстур

Тип текстуры Рекомендуемый формат

MRAO / Specular DXT1
